# Contributing to Elyan Labs Site

Welcome! Here's how to contribute.

## Getting Started

1. Fork this repo
2. Clone: `git clone https://github.com/YOUR_USERNAME/elyan-labs-site.git`
3. Create branch: `git checkout -b fix-description`

## Making Changes

- Edit the relevant files
- Test your changes locally
- Commit with clear messages

## Submitting

1. Push to your fork
2. Open a Pull Request
3. Reference any related issues

## Questions?

Open an issue for discussion.

---

*Thank you for contributing! 🚀*
